# Efektivní kódy
A2TPI Semestrální projekt č. 4

## Popis
Program je určen pro kódovanými textu pomocí metody efektivního kódovaní. Program umožnuje kódovat text Shannon–Fanovu metodu nebo metodu Huffmanovu.
